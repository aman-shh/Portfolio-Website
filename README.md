# Personal Portfolio Website

A modern and responsive personal portfolio website built using HTML5, CSS3, and JavaScript.

## Overview

This portfolio showcases my background, technical skills, project work, and contact information in a clean and professional interface.

The website features:

- Responsive design for desktop and mobile devices
- Smooth scrolling navigation
- Scroll reveal animations
- Interactive navigation highlighting
- Modern dark-themed UI
- Structured sections for About, Skills, Projects, and Contact

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
